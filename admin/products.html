Модерация карточек

Карточки, созданные менеджерами, скрыты во витрине до одобрения. Уведомление в Telegram уходит только на ID из переменной окружения сервера OWNER_TELEGRAM_IDS (главный администратор).

Сейчас нет карточек на модерации.

Доступ менеджеров

Назначьте флаг is_staff по Telegram ID — в профиле мини-приложения появится пункт «Панель управления». Пароль для входа менеджеров в эту веб-админку задаётся отдельно в настройках сервера (файл окружения на хостинге), не в интерфейсе. Эта панель назначения прав отображается только после входа основным паролем владельца из тех же настроек сервера. Снять доступ менеджера с закреплённых аккаунтов владельца нельзя — список задаётся при настройке сервера.
